PROJECTS

Protecting Cloud Data With Dynamic Security Via Network Coding

This project secures cloud data using dynamic security techniques with network coding in Java, JDBC, and MySQL. It encrypts data before storage and dynamically updates security keys to prevent unauthorized access. Java handles encryption and data transmission, JDBC connects to MySQL for secure storage, and network coding enhances data integrity and confidentiality.

TICTACTOE GAME

Tic-Tac-Toe is a simple two-player game built using HTML, CSS, and JavaScript. The game board consists of a 3×3 grid where players take turns marking "X" or "O" in empty cells. The objective is to form a horizontal, vertical, or diagonal line with the same symbol to win. HTML structures the board, CSS enhances its appearance with styling, and JavaScript manages game logic, including turns, win detection, and resets. Players receive real-time feedback, and the game can reset after a win or draw. It’s a great beginner project to learn DOM manipulation, event handling, and basic game logic.

Student Management System

The Student Management System is a web-based application developed using Java, JDBC, HTML, CSS, JavaScript, and MySQL, deployed on a Tomcat server. It enables administrators to manage student records, including registration, login, and academic details. The system follows the MVC architecture, utilizing Servlets for backend processing, JSP for dynamic content, and MySQL for data storage. JavaScript ensures client-side validation, enhancing user experience and data integrity.
